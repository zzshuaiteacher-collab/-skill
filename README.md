## 🚀 如何使用？




1.  **激活视角**：在对话中输入对应的触发词。
    *   *示例：* 「切换到马云视角，分析下我现在的商业困惑。」
2.  **创建新大脑**：如果你有女娲工具，可以继续使用其指令创建新视角。
