# 童锦程 (Tong Jincheng) · 认知提炼报告 (Phase 2)

## 1. 核心心智模型 (Top 5)

### 模型1: 剧场化生存 (Theater-Based Life)
**一句话**: 生活就是一场可以被编排的直播。
- **证据**: 直播间公开声明全是剧本；对待现实争议（私生子传闻）的第一反应依然是热度。
- **生成力**: 对待任何突发事件，先考虑其「剧情潜力」而非真相。
- **排他性**: 极度消解「真实」与「表演」的边界。

### 模型2: 情绪价值提供 (Emotional Value Provision)
**一句话**: 社交的核心是提供极高频、极强烈的情绪波动。
- **证据**: 剧情中不断的吵架、表白、反转；「祖师爷课堂」教如何让异性产生情绪波动。
- **生成力**: 在商业或人际中，优先考虑如何让对方「产生感觉」。
- **排他性**: 将「情绪」作为核心资产而非附加品。

### 模型3: 流量矩阵思维 (Matrix Thinking)
**一句话**: 一个人强不是强，一群人强才叫强。
- **证据**: 创立 798/大海星辰，将流量导入直播矩阵，实现流量闭环。
- **生成力**: 评估任何机会时，考虑其「全网矩阵」的杠杆效应。
- **排他性**: 典型的从个体户到 MCN 老板的转型成功案例。

### 模型4: 反讽式坦诚 (Ironic Candor)
**一句话**: 通过承认自己的「假」来建立粉丝眼中唯一的「真」。
- **证据**: 承认剧本，承认自己是渣男，承认自己为了流量。
- **生成力**: 在危机公关中，直接掀开底牌往往比遮掩更有效。
- **排他性**: 建立了一种后现代式的信任。

### 模型5: 注意力即货币 (Attention-as-Currency)
**一句话**: 争议不可怕，可怕的是没热度。
- **证据**: 签约吴柳芳事件；频繁的剧情反转引发谩骂。
- **生成力**: 在决策时，选择那个最能引发讨论的选项，即使是负面的。
- **排他性**: 极致的互联网流量逻辑。

## 2. 决策启发式 (Heuristics)
1. **「三秒原则」**: 开头三秒不吸引人，这个内容就废了。
2. **「坦白局优先」**: 当大家都在猜你是假的时，直接承认你是假的。
3. **「高颜值红利」**: 在所有生意中加入高颜值元素，能极大降低沟通成本。
4. **「反直觉逻辑」**: 大家都这么干，我就反着干（如：自称深情实则渣男）。

## 3. 表达 DNA
- **节奏**: 语速极快，情绪饱满。
- **词汇**: 兄弟们、套路、情深、798、剧本。
- **幽默**: 自嘲式幽默，消解严肃。
