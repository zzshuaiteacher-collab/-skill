# 调研报告：马云的著作与系统性表达 (01-writings.md)

## 1. 核心著作
| 书名 | 年份 | 核心论点/贡献 |
| :--- | :--- | :--- |
| 《马云内部讲话》 | 2010 | 强调阿里的使命感、价值观（六脉神剑）、「客户第一、员工第二、股东第三」。 |
| 《未来已来》 | 2017 | 提出从 IT 到 DT（Data Technology）的转型，定义「五新」（新零售、新制造、新金融、新技术、新资源）。 |
| 《马云：我的世界观》 | 2015 | 商业应基于利他主义，探讨全球化、互联网生态。 |

## 2. 关键概念与自创术语
- **DT 时代 (Data Technology)**：认为未来是基于数据的技术时代，强调赋能而非控制。
- **五新**：新零售、新制造、新金融、新技术、新资源。其中「新零售」流传最广，强调线上线下融合。
- **eWTP (Electronic World Trade Platform)**：倡导建立全球电子商务平台，帮助中小企业参与全球贸易。
- **亩产一千美金**：针对农村淘宝提出的农业数字化目标。

## 3. 智识谱系与影响力
- **金庸武侠文化**：阿里早期的「六脉神剑」价值观、办公区命名、员工花名均源于此。
- **道家/太极哲学**：强调平衡、演化、生态，而非简单的机械式增长。

## 4. 矛盾与张力
- **规模与小而美**：一方面将阿里做成巨头，另一方面反复强调要帮助「小企业」。
- **技术与人文**：虽然做技术驱动的公司，但自称不懂技术，更强调人的价值和使命。
