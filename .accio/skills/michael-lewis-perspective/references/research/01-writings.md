# 调研报告：迈克尔·刘易斯的著作与核心论点 (01-writings.md)

## 1. 核心著作与主题
| 书名 | 年份 | 核心论点/贡献 |
| :--- | :--- | :--- |
| 《说谎者的扑克牌》 (*Liar's Poker*) | 1989 | 揭露华尔街 80 年代疯狂的交易文化，展示了金钱如何异化人性及系统性风险的萌芽。 |
| 《点球成金》 (*Moneyball*) | 2003 | **效率模型**：传统直觉在面对大数据时的溃败。展示了小球会如何利用统计学（Sabermetrics）打败资本巨头。 |
| 《弱点》 (*The Blind Side*) | 2006 | **价值发现**：一个被社会遗忘的人如何通过正确的坐标系找到自己的位置。 |
| 《大空头》 (*The Big Short*) | 2010 | **局外人优势**：当整个系统都在疯狂时，只有那些处于边缘、不合群的人才能看清真相。 |
| 《第五风险》 (*The Fifth Risk*) | 2018 | **管理风险**：政府中那些处理「看不见的风险」的无名英雄，以及忽视这些风险的代价。 |
| 《走入无限》 (*Going Infinite*) | 2023 | 关于 SBF 和 FTX 兴衰，探讨了加密货币圈的异化及由于个人特质导致的系统性崩溃。 |

## 2. 关键心智模型与方法论
- **局外人视角 (The Outsider's Advantage)**：认为系统的参与者往往看不清系统，只有边缘观察者才能发现结构性错误。
- **叙事弧线 (Narrative Arc)**：将复杂的经济/科学问题「人格化」，通过一个人的旅程来解释宏观现象。
- **效率 vs 直觉 (Efficiency vs Intuition)**：这是其一贯的主题，从棒球到政府管理，都在探讨系统如何因为迷信直觉而产生低效。
- **看不见的风险 (The Hidden Risk)**：强调那些虽然概率极低但一旦发生就具有毁灭性的风险。

## 3. 推荐书单/智识谱系
- 深度受**行为经济学**影响（如 Daniel Kahneman 和 Amos Tversky 的研究，曾专门写过《思维的发现》）。
- 偏好那些能提供「第一性原理」解释或颠覆传统认知的非虚构作品。
