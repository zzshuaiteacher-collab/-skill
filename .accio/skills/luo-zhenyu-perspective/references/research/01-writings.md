# 调研报告：罗振宇的著作与知识逻辑 (01-writings.md)

## 1. 核心著作与节目
| 书名/节目 | 核心论点 |
| :--- | :--- |
| 《罗辑思维》(节目) | 早期通过「死磕」自己，通过知识套利，将晦涩的社科、历史、经济知识转化为普通人能听懂的逻辑故事。 |
| 《时间的朋友》(跨年演讲) | 提倡「长期主义」，发愿连续举办 20 年。强调在不确定性中通过建立长期确定性的逻辑来对抗焦虑。 |
| 《知识服务》相关方法论 | 认为知识不仅是内容，更是一种服务。主张「替用户读书」，将知识「胶囊化」、「碎片化」。 |
| 《阅读方法》 | 提倡「目的性阅读」，将书作为解决现实问题的工具箱。 |

## 2. 关键心智模型与自创术语
- **知识胶囊/脱水知识**：将庞杂的知识压缩成可被快速消化的颗粒。
- **时间的朋友 (Long-termism)**：在波动中寻找不变的价值，通过复利效应实现增长。
- **认知升级**：认为人与人之间唯一的壁垒是认知，商业竞争本质上是认知竞争。
- **小趋势**：关注那些还没被大众察觉、但足以改变局面的细微变量。
- **存量思维 vs 增量思维**：在增量消失时代，如何通过深耕存量资产实现突围。

## 3. 智识谱系
- 深度受**凯文·凯利 (KK)** 的失控理论、**德鲁克**的管理学思想影响。
- 将自己定位为「知识搬运工」或「知识翻译官」。
