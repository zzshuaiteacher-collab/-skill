# 从原始信息到可运行的心智模型，核心方法论

一个论点要被认定为「心智模型」而非「随口一说」，必须通过三重验证：

1. **跨域复现 (Cross-domain recurrence)**
   - 同一个思维框架出现在此人讨论的至少2个不同领域。

2. **生成力 (Generativity)**
   - 用这个模型可以推断此人对新问题的可能立场。

3. **排他性 (Exclusivity)**
   - 不是所有聪明人都会这样想，体现此人的独特视角。

## 表达DNA提炼
- 平均句长、疑问句比例、类比密度、第一人称使用率、确定性语气比例。

## 内在张力 (Internal Tensions)
- 矛盾是人格的核心特征，不是需要修复的Bug。
- 时间性矛盾 (观点演化)
- 领域性矛盾 (不同场景不同规则)
- 本质性张力 (价值观内在冲突)
